# Duck Bros

A Unity game project for Duck Bros.

## Project Structure

- `Assets/` - Game assets, scripts, scenes, prefabs, and settings.
- `Packages/` - Unity package dependencies.
- `ProjectSettings/` - Unity project configuration.

## Getting Started

1. Install **Unity Hub**.
2. Open Unity Hub and add this folder as a project.
3. Open the project with the Unity Editor version specified in `ProjectSettings/ProjectVersion.txt`.
4. Open a scene from `Assets/Scenes/` and press Play.

## Version Control Notes

- The repository includes Unity `.meta` files.
- Generated folders like `Library/`, `Logs/`, and `UserSettings/` are excluded via `.gitignore`.
